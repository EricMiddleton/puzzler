## Structure

Uses opencv release 3.2.0.

## Setup:
1. clone into desired directory

`git clone https://git.ece.iastate.edu:luehm/puzzler`

2. update submodule

`cd puzzler/src`
`git submodule init`
`git submodule update`

3. make opencv libraries

`make opencv -B`
